## 👩‍🏫활동 소개

문화내러티브학과 게임동아리에서 게임을 기획

🗓️ **작업기간** : 2016/11/01 → 2016/12/28

👨‍💻 **투입인원** : 6명

📒 **주요업무** 

- 게임 시작 화면, 중간 화면, 엔딩을 스토리라인에 맞춰서 기획
- 플레이어의 선택에 따라서 게이지가 쌓이고 그에 따라 다른 엔딩들을 기획
- 게임 이미지 및 캐릭터 대사 작성
